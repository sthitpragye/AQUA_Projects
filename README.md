## Project 1: Macro-Factor Driven Sector Allocation

Repository: https://github.com/sthitpragye/AQUA_Project_1_Macro_Factors

### Objective 

Build a macro-driven sector allocation framework for Indian equities by combining market regime detection with macroeconomic sensitivities.

### Key Features : 

- Hidden Markov Model (HMM) based market regime detection
- VECM and FEVD based macro-factor analysis
- Sector allocation engine
- Dynamic portfolio weighting
- Macro shock adjustments
- Trade signal generation

### Methodology : 

1. Detect market regimes (Bullish / Bearish / Sideways)
2. Estimate macro sensitivities using:
   - Oil prices
   - USD/INR exchange rates
   - FII flows
3. Allocate sector weights
4. Generate portfolio signals

## Project 4: Options Greeks

Repository: https://github.com/sthitpragye/AQUA_Project_4_Greeks

### Objective

Build a regime-aware options backtesting framework for NIFTY 50 using Greeks and ML-based market detection.

### Key Features

- Black-Scholes Greeks engine for pricing and risk management
- HMM + GMM regime detection (Trending / Range-Bound / Stress)
- 4 regime-gated strategies (Iron Condor, Credit Spread, Straddle, Bull Put Dip)
- Walk-forward optimization to avoid look-ahead bias
- 72.1% win rate, 2.34 profit factor, -1.85% max drawdown
